# Поиск анаграмм в словаре

Программа на языке Go для поиска всех множеств анаграмм в заданном списке слов на русском языке.

Анаграммы — это слова, состоящие из одинаковых букв в разном порядке  
(например: *«пятак»*, *«пятка»*, *«тяпка»*).

---

## Ключевые особенности:

- Поддержка слов на русском языке (Unicode).
- Автоматическое приведение всех слов к нижнему регистру.
- Группировка слов по анаграммам.
- Исключение одиночных слов (без пар).
- Отсортированный вывод слов внутри каждой группы.
- Эффективная работа даже с большими словарями.

---

## Вход и выход

**Вход**: срез строк (`[]string`)

**Выход**: `map[string][]string`, где:
- **ключ** - первое встреченное слово из группы анаграмм,
- **значение** - отсортированный список всех слов из этой группы.

> Группы из одного слова **исключаются** из результата.

### Пример

```go
words := []string{"пятак", "пятка", "тяпка", "листок", "слиток", "столик", "стол"}
result := FindAnagrams(words)
```

Возможный результат:
```go
map[string][]string{
    "пятак":  {"пятак", "пятка", "тяпка"},
    "листок": {"листок", "слиток", "столик"},
}
```

Слово `"стол"` отсутствует - у него нет анаграмм.

---

## Алгоритм

1. Каждое слово приводится к нижнему регистру.
2. Буквы в слове сортируются - получается **нормализованный ключ** (например, `"пятак"` → `"акпты"`).
3. Слова группируются по этому ключу.
4. Для каждой группы запоминается **первое встреченное слово** как ключ в финальном результате.
5. Группы из одного слова отбрасываются.
6. Слова в каждой группе сортируются по алфавиту.
